Telco Customer Churn Prediction

## Project Overview
This project uses machine learning to predict whether a telecom customer will churn (stop using the service). The goal is to help the company retain at-risk customers.

## Dataset
- **Source**: IBM Telco Customer Churn Dataset
- **Rows:** 7,043 customers
- **Features:** 21 columns (demographics, services, account info)

## Models Used
- Logistic Regression
- Random Forest
- XGBoost

## Evaluation Metrics
- Accuracy
- Recall
- ROC-AUC

## Results
| Model | Accuracy | Recall | ROC-AUC |
| Logistic Regression | 0.786 | 0.494 | 0.830 |
| Random Forest | 0.792 | 0.487 | 0.813 |
| XGBoost | 0.764 | 0.487 | 0.808 |
